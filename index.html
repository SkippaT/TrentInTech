Interactive NFC Music and LED Controller

This project uses a Raspberry Pi computer to control music using the Spotify API. Connected to the Raspberry Pi is an NFC scanner which reads my 'Music Cards' and performs a certain action depending on what card was used.

Also connected to the Raspberry Pi is an LED strip that is used for feedback and because LEDs are awesome!

Equipment

  • Raspberry Pi 3 Model A+
    • The brains of the operation
  • WS2812B LED strip
    • 5 metres with a total of 150 LEDs
  • NFC HAT for Raspberry Pi (PN532)
    • The scanner for the NFC cards
  • 13.56MHz NFC key cards
    • The cards that control what the Raspberry Pi does

Process

  1. I began this project by first playing with the NFC scanner. I created programs that waited for a card to be scanned and then printed the UID to the terminal.
  2. I next researched the Spotify API and created a program that takes user input from the terminal to control my Spotify. This program could pause and skip songs, play a playlist, album or song, toggle shuffle, adjust volume and many other things.
  3. I then combinded the two programs I'd made. This meant I was controlling Spotify with the NFC cards.
  4. The next step was creating some functions for the LED strip to perform. Some of these functions are purely for show and others are for feedback after a card is pressed.
  5. With the application working properly, the final step was to install the device in my room. To do this, I 3D printed a mount for the NFC scanner underneath my bedside table. The LEDs were attached around the perimeter of my bed out of sight.
